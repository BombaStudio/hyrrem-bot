### How to setup this



#### 1-) install these packages

```shell
python3 -m pip install -r requirements.txt
```

#### 2-) Edit config.json

```json
{
    "BotToken":"Your bot token",
    "AdminID":"Your user ID"
}
```

#### 3-) Run this code

```python3
python3 main.py
```
